+--------------------------------------------------------------------------
|  1. Installation
+--------------------------------------------------------------------------

To get a copy of this code:

  $ svn co http://deskcorder.googlecode.com/svn/trunk ./deskcorder
  $ cd deskcorder
  $ python deskcorder.py

Enjoy!



+--------------------------------------------------------------------------
|  2. Complaining
+--------------------------------------------------------------------------

Go to http://code.google.com/p/deskcorder/issues/list and file a new issue
with the "new issue" link.

If you're submitting a bug report, please be sure to include all useful
information to help us recreate the error, so that we can fix it quickly.

If you're submitting a bug fix, please be sure to add a patch file and
describe the bug you're fixing.
